# Grupos

Los grupos son una clase que te permite agrupar objetos, esto permite orden:

```haxe
import vupx.groups.VpGroup;

var group = new VpGroup();

// Agregar un objeto al grupo
var sprite = new VpSprite(100, 100);
group.add(sprite);
```
